# Drop Frontend

This is the frontend application for the Drop E-Commerce platform. It is built using [Next.js](https://nextjs.org/) and leverages the App Router for optimal performance, server-side rendering, and static site generation.

## Features

- **Modern Stack**: React 19, Next.js 16, TypeScript
- **Styling**: Tailwind CSS integration for rapid and responsive UI development
- **Animations**: Framer Motion for smooth interactions and transitions
- **Icons**: Lucide React for consistent and crisp vector icons
- **Data Fetching**: Axios for robust API interactions
- **Linting**: ESLint integrated to ensure code quality

## Quick Start

```bash
# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
